This is an anrdoid application written by Erik Gustafson (gustafson.93@osu.edu) for 
The Ohio State Student radio station, Arouse. In order to run this project you must
import it into Android studio, build the .apk file using Gradle, and load the 
executable to a mobile device or android emulator via ADB. This project is occasionally still 
in development and can be found here: https://github.com/gusty9/OSUArouseRadio
